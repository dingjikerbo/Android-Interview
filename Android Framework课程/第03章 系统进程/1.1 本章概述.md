# 本章概述

### 问题：按下手机电源键后发生了什么？

本章将围绕这个问题展开，分5个小节来讲述，跟android启动相关的内容

 - zygote
 - system server
 - ServiceManager
 - system service
 - launcher

这些都和Android应用的基础，正因为有这些组件的启动，才能为应用提供全方位的支持。 

比如zygote开辟应用进程
system server中跑了很多系统服务
servicemanager为Binder
system service，AMS，PMS等，四大组件才能跑起来，才能调动系统的各类资源
launcher桌面就不说了，用户面对的应用程序的总入口
